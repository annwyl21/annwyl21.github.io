Yahtzee Scorer Demo

  • To play you need 5 dice
  • On your turn roll the dice and record your roll
  • Your roll will be sent to the score calculator class
    and will return an enumerated result with the possible scores
  • select the number that corresponds to your score
  • Play 13 rounds, bonuses are applied at the end
  • The scoreboard object holds the scores throughout the game

This demo is to demonstrate my knowledge of object-oriented programming, developing 2 objects: the scoreboard class and the calculator class. The main purpose for the development was to build the scorer object, the scoreboard allows you to see that my core logic works.

GitHub repo also contains the 43 tests I wrote in pytest and pextract when I used Test Driven Development to write my code. The computer is not designed to play competitvely.
