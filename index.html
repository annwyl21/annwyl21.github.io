Github Repo: symptom_record

The current project utilizes a metaprompt prefix in API requests to the text-davinci-003 AI model, summarizing user-logged symptoms. These symptoms, timestamped and stored, can be summarized on demand. A bubbleplot illustrates the frequency of symptoms over time and throughout the day, with bubble size signifying reported pain levels.

To develop this application, I acquired knowledge in NLP and the NLTK, creating a preliminary summarizing tool to establish a benchmark for the API/metaprompt development.
